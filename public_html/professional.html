Simple SQL Injection Demo

This is an example of one of the most simple SQL (Standard Query Language) injection payloads.
SQL injection submits the payload into the database on the back-end to query for data to be accessed that otherwise
shouldn't be available.
This payload uses ' OR 1=1-- for the username and ' for the password to bypass the login page.

Breakdown:
' uses a single quotation to close any string literals.
OR 1=1 uses conditional logic to always return true, returning all rows.
-- uses to dashes to comment out everything after the dashes, nullifying the execution of any later statements, therefore ignoring everything after.
